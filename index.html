Research Domains

Active fields of research and development across the doctoral group.

Control Theory
Adaptive & optimal control (LQR/MPC), plant modelling, stability analysis, H∞ methods
Robotics
SLAM, ROS2 architecture, kinematic models, autonomous navigation, collaborative systems
Computer Science
Low-level programming, firmware, backend systems, system architecture, OS internals
Symbolic AI
Knowledge representation, semantic webs, expert systems, logic programming, ontologies
Sub-Symbolic AI
Neural networks, deep learning, fuzzy logic, evolutionary algorithms, reinforcement learning
Data Science
EDA, feature engineering, causal inference, anomaly detection, time-series analysis
Digital Twins
Real-time synchronisation, physics-based simulation, IoT ingestion, IBM Twin frameworks
ERP / IRP
IBM IRP: resource integration, process orchestration, OT/IT convergence, AI-driven planning
Industrial Engineering
Industry 5.0, human-machine collaboration, predictive manufacturing, future systems analysis

Home Brew

Open source projects built by the group. Sorted by contributors.

PIDLab 11

Browser-based interactive PID controller simulator. Tune gains in real time and observe step response, Bode plots, and root locus with live feedback. Built for students and educators.

PID Simulation Browser Education
JavaScript WebAssembly SVG
petrova
OpenCTRL 7

Open-source PID tuning and analysis toolkit for embedded targets. Supports live parameter sweeps, step-response visualisation, and direct export to C header files ready for MCU flashing.

Embedded Control Theory PID Tooling
C++ Python CMake
karageorgiev
NeuroFuzz 4

Fuzzy-neural hybrid inference engine for real-time classification on resource-constrained microcontrollers. Targets STM32 and ESP32. Includes a training pipeline and a model-to-C-array compiler.

Neural Networks Fuzzy Logic Embedded MCU
Python C PyTorch
medev
